Manage Your Tasks.

This is a simple To Do List website where you can create to do lists to organise and manage the tasks in your life

About this website...


This website was created by Kiera Walton as part of a QA Academy project.

Here, you can create to do lists and manage the tasks in those lists - the details of which will be stored on a locally hosted database.

Select CRUD lists to create, read, update or delete to do lists.


Functionality of this website:
  • Create lists and add tasks to them
  • Read all lists stored on the database
  • Read lists by ID
  • Update list and task details
  • Delete lists by ID
